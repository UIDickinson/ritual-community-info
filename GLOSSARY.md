# Ritual Community — Glossary of Terms

| Term | Meaning |
|------|---------|
| **Gritual** | The community greeting. Used to say hello (e.g., "Gritual mate!", "Gritual champ!"). |
| **NPC** | Non-Player Character — refers to the bot army that attacked the Ritual community during the NPC War / 21-day challenge. Also used to playfully insult someone acting mindless. |
| **Ritual / The Ritual** | Refers broadly to the Ritual Network community, its goals, and its ecosystem. |
| **Ritualist** | An active, contributing community member who has earned the Ritualist role. |
| **Syns** | Sins — the community spells it "syns." Used in the Confessions and Blessings of Syn channels. |
| **Yapping** | Active chatting / contributing in the community. Positive connotation. |
| **Blessings** | Positive reputation points given by community members via the Vestibule channel. |
| **Curses** | Negative reputation points given by community members via the Vestibule channel. |
| **Wen** | Community meme — short for "when" (as in "wen token?", "wen airdrop?"). The Wen channel is dedicated to this. |
| **Pledge / Pledging** | Committing to the Ritual community (Blessings of Syn channel). Earning the Ascendant role. |
| **The Void** | The time before Ritual existed. Forerunners "made it through the void." |
| **Dunce** | A humorous insult role for those "without intelligence." |
| **Forerunner** | An OG — someone from the earliest days of Ritual, before the current community formed. |
| **Mage** | An exceptional creative contributor (artist, content creator). |
| **Radiant** | The highest community role — Radiant Ritualist. Super rare. |
| **Ritty** | Loyal long-term member. Has access to exclusive Telegram chat. |
| **Ritty Bitty** | A baby ritualist — on the right path but still growing. |
| **Harmonic** | A balanced state — equal blessings and curses. |
| **Node Runner** | Historical role for early infrastructure operators (no longer available). |
| **PFP** | Profile picture — several events revolve around custom PFP creation. |
| **Ez** | "Easy" — used sarcastically after winning (or losing) a game. |
